<ins>Organización de la estructura del repositorio </ins>

El repositorio cuenta con una organización que comienza según este orden:

    -Clases 
    -CSS
    -HTML
    -Java
    -PHP
    -Practicos
    -Proyecto
    -Temas
    -Teoricos
    -Trabajos_en_casa

El directorio 'Clases' representa todo el material que demos en clase, 'HTML Y CSS' todo los archivos .css y .htmlque yo cree, lo mismo con HTML, Java y PHP, 'Practicos' es para guardar los practicos en ese directorio, 'Proyecto' el proyecto, 'Temas' los temas que demos en clase, 'Teoricos' los teoricos, 'Trabajos_en_casa' los trabajos que realizare en casa.

Este sistema fue pensado para tener el mayor orden posible.
